DARKSTAR-NOTINPKG(1)             General Commands Manual (darkstar-tools-14.2)

NAME
     darkstar-notinpkg and darkstar-notonsys â€“ compares file list between
     files on the system and files that are installed according to the package
     database

SYNOPSIS
     darkstar-notinpkg
     darkstar-notonsys

DESCRIPTION
     darkstar-notinpkg prints files that are present on the / filesystem that
     doesn't belong to any package.

     darkstar-notonsys prints files that have been installed with some
     package, but aren't present on the system.
